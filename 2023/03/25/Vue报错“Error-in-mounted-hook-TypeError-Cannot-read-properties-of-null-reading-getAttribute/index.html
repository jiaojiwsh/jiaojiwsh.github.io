# 问题描述

在搭配项目配置 Echarts 图表时，运行项目开发者工具报出 “Error in mounted hook: "TypeError: Cannot read properties of null (reading ‘getAttribute’)" 错误，并且图表不显示，具体如下：

# 解决方法

查看代码，并没有发现什么异常

将代码使用控制台打印，能正常显示

控制台能够一时间打印出来，查看地图配置区域，发现是 id 值，而自己 div 里给的是 class，结果调用的也是 class 值

将 class 改为 id, 这是图表就可以显示出来啦
